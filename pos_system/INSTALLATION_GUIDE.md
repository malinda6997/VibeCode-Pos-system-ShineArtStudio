# Shine Art Studio POS - Installation Guide

## Quick Desktop Shortcut (Easiest Method)

### Option 1: Using the Batch File

1. Go to: `F:\2025 NEW PROJECTS\Pasindu\Shine Art Studio\pos_system\dist\`
2. Double-click `Create_Desktop_Shortcut.bat`
3. A shortcut will appear on your desktop
4. Done! Click the desktop shortcut to run the POS system

### Option 2: Manual Desktop Shortcut

1. Go to: `F:\2025 NEW PROJECTS\Pasindu\Shine Art Studio\pos_system\dist\`
2. Right-click on `ShineArtStudio_POS.exe`
3. Select "Send to" → "Desktop (create shortcut)"
4. Done! Use the desktop icon to launch

---

## Professional Installer (Using Inno Setup)

If you want a professional Windows installer:

### Step 1: Install Inno Setup

1. Download from: https://jrsoftware.org/isdl.php
2. Install Inno Setup (free)

### Step 2: Create Installer

1. Open Inno Setup Compiler
2. File → Open → Select: `F:\2025 NEW PROJECTS\Pasindu\Shine Art Studio\pos_system\installer_script.iss`
3. Click "Build" → "Compile"
4. The installer will be created: `ShineArtStudio_POS_Installer.exe`

### Step 3: Run Installer

1. Double-click `ShineArtStudio_POS_Installer.exe`
2. Follow installation wizard
3. Choose to create desktop icon (recommended)
4. Installation complete!

The program will be installed to: `C:\Program Files\ShineArtStudio\`
